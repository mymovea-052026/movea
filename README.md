## Installation locale

1. Installer XAMPP
2. Cloner ce dépôt dans `C:\xampp\htdocs\movea`
3. Démarrer Apache via XAMPP Control Panel
4. Ouvrir `http://localhost/movea` dans le navigateur

## Pile technique prévue

- Backend : PHP 8.2
- Base de données : MySQL 8 (avec extensions spatiales)
- Cache & temps réel : Redis
- Frontend : HTML5, CSS3, JavaScript (vanilla puis framework léger)
- Cartographie : Leaflet + OpenStreetMap (puis Google Maps si nécessaire)
